The Belgian Berry – Artificial Table Top Christmas Tree – Green – 3 foot

“The Belgian Berry” 3ft (90 cm) is a hinged style table top Christmas tree.  In this model, our Belgian’s mixture of PVC and PE branches have been pre-decorated with traditional red berries and pine cones which combined with the fact that this is a hinged Christmas tree, result in an easy “all-in-one” solution requiring no further decorations (unless you would like to light it up).
